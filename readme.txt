Enables bulk plugin installation for the WP Plugin Manager Chrome Extension.

== Description ==

This helper plugin works with the WP Plugin Manager Chrome Extension to enable:

* Bulk plugin installation
* Automatic plugin activation
* Fast, efficient plugin management
* REST API and AJAX fallback support

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-plugin-manager-helper` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use with the WP Plugin Manager Chrome Extension
